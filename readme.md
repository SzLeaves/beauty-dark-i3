# beauty-dark-i3
A **dark** theme about i3wm, include dunst, picom, polybar and rofi.  

## dependency
* i3wm (or i3-gaps)
* dunst
* picom (or compton)
* polybar
* rofi
* xfce4-power-manager (used to backlight adjust)
* terminator
* pulseaudio
* nmcli

## Location
`$HOME/.config/i3`
